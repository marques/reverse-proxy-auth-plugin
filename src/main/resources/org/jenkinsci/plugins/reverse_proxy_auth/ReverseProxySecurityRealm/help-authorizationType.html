Selects the authorization type that will be used:
  • Header groups: the default option. If this option is selected then the user groups are obtained, if they are provided, from HTTP headers.
  • Crowd: user groups are obtained via Crowd server. If this option is selected, then the Crowd authorization properties should be provided. Crowd server, application name and application password are mandatory to properly configure Crowd access.
  • LDAP: user groups are obtained via LDAP. If this option is selected, then the LDAP authorization properties should be provided.
